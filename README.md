> vuex 3.1.2  
> element-ui 2.13.0  
> vue-router 3.1.3  
> vue-cli 4.1.2  
> chart.js 2.9.3  
> sortablejs 1.10.2  

## 安装步骤

``` 
git clone https://github.com/ZengKunLun/vue-elementui-vuex-admin.git //下载源码
cd vue-elementui-vuex-admin //进入文件夹
cnpm install //安装项目依赖

npm run serve //开启服务器，浏览器访问 http://localhost:8000
npm run build //生成的dist文件夹放在服务器下即可访问
```
